Hytale è progettato per supportare una varietà di metodi di input per soddisfare le diverse preferenze e piattaforme del giocatore.

## Sistema Di Interazione

Hytale è dotato di un solido sistema di interazione che consente ai plugin di personalizzare il modo in cui vengono interpretati i tasti di scelta rapida e le azioni che essi attivano. Questo sistema è data-driven ("*guidato dai dati*") e consente ai modder di creare interazioni complesse attraverso le risorse.

I modder possono definire tasti personalizzati per varie azioni, come l'uso di abilità o l'interazione con oggetti nel mondo di gioco. Il sistema di interazione è progettato per essere potente, anche se può richiedere un certo sforzo per gestire la complessità delle risorse coinvolte.

> Puoi usare le proprie interazioni per assegnare i tasti di scelta rapida a qualsiasi funzione si desideri. Il sistema di interazione è molto potente, ma dobbiamo ancora lavorare sulla sua usabilità in quanto gli asset possono diventare abbastanza complessi e difficili da gestire
