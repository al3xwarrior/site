## Kenapa belajar coding? Kita kan punya visual scripting!

Ya memang, dan sejujurnya kamu harus mencobanya. Visual scripting adalah tambahan yang hebat untuk Hytale dan kami tahu itu akan sangat kuat, tapi tidak ada yang mengalahkan kemampuan untuk membaca dan menulis kode Java, dengan kemampuan untuk membangun mod dan plugin yang jauh lebih kuat.

<Callout type="info" title="Visual Scripting vs Coding">
  Visual scripting is great for beginners and for simple tasks, but coding allows for more complex and efficient solutions. Learning to code opens up a wider range of possibilities for modding Hytale.

  We will have guides on visual scripting when Hytale releases from Day 1.
</Callout>

## Haruskah semua orang yang mencoba modding Hytale belajar coding?

Tidak ada yang boleh merasa bahwa mereka HARUS belajar coding untuk membuat sesuatu di Hytale. Simon dan timnya telah membuktikan hal ini berkali-kali dengan usaha mereka dalam editor grafik dan visual scripting. Kamu bisa mengubah perilaku mob, generasi dunia, dll. semuanya hanya dengan alat-alat tersebut.

Tapi jika kamu ingin membangun hal-hal yang lebih kompleks (mulai dari utilitas hingga perangkat lunak server yang sepenuhnya kustom) yang bisa digunakan oleh jaringan server skala besar dan nama-nama besar, kamu pasti ingin belajar coding. Dan di situlah panduan ini berperan...

## Oke, saya tertarik. Apa selanjutnya?

Over the course of these next several lessons you'll learn the basics of coding, how to read documentation, and how to find resources to help you learn more on your own.

The guides themselves will take a very big "bird's-eye view" approach to things and provide links to resources to do the actual coding. Hal ini dilakukan karena, jujur saja, sudah ada cukup banyak sumber daya tentang cara mempelajari hal ini, hanya saja mencari hal yang tepat di Google bisa jadi sulit. Belajar untuk bisa coding tanpa bimbingan adalah tujuannya. Komunitas Modding Hytale selalu ada untuk membantumu, tapi untuk menjadi pengembang yang benar-benar matang, kamu harus mempelajari banyak hal ini hanya dengan membaca dari sumber daya yang ada.

<Callout type="info" title="Finding Resources">
  We have created a small Java Basics series to help you get started with the language itself, but beyond that, you'll be expected to find your own resources to learn more. We recommend sites like [w3schools](https://www.w3schools.com/java/) and [GeeksforGeeks](https://www.geeksforgeeks.org/java/) as great starting points.
</Callout>

Our goal is to teach you how to Mod Hytale, and that will always be our goal. This is the reason we can't teach you everything about Java or programming in general. Instead, we want to give you the tools to be able to learn on your own, so you can continue to grow as a developer even after you've completed our guides.

## Ada kata penutup?

Ingat, belajar coding itu sulit. Kamu akan tersandung dan jatuh, tapi kamu akan bisa bangkit kembali. Tetaplah tangguh, percayalah kamu bisa membuat hal-hal keren, dan aku berjanji kesuksesan akan datang. Happy coding!
