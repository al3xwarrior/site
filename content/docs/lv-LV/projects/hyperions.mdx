# Hyperions

<Callout>
  Šis ir kopienas veidots projekts, ko uzsāka lietotājs [MaeMachineBroke](https://github.com/maemachinebroke).
</Callout>

**Hyperions** ir Hytale API bibliotēka, kas paredzēta, lai nodrošinātu citus mod ar API enerģijas uzglabāšanai, pārraidei un saņemšanai. Bibliotēka izmanto kopienas dalībnieka pārsūtīšanas API shailist {/* TODO link */} un ir bāzēts uz [Energy](https://github.com/TechReborn/Energy) API no TechReborn komandas.

Standarta enerģijas vienību, ko izmanto Hyperions, sauc par Hyperion (saīsināti `Hy`), un viens Hyperion ir aptuveni vienāds ar {/* TODO */}. Standarta pārneses ātruma vienība ir Hyperions katru tikšķi ("Hy/t") vai alternatīvi dalot ar 30 iegūtu hiperjonus sekundē ("Hy/s").
