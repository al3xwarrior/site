# Hyperions

<Callout>
  Este é um projeto criado pela comunidade e iniciado por [MaeMachineBroke](https://github.com/maemachinebroke).
</Callout>

A **Hyperions** é uma biblioteca de API para Hytale projetada para fornecer armazenamento, transmissão e recebimento de energia para outros mods. A biblioteca usa a Transfer API do membro da comunidade [shailist](https://github.com/shailist) sendo baseada na API [Energy](https://github.com/TechReborn/Energy) da equipe TechReborn.

A unidade padrão de energia usada pelo Hyperions é chamada de Hyperion (abreviação: `Hy`) que equivale a ~{/* TODO */}. A unidade padrão para a taxa de transferência é o Hyperion por tique (`Hy/t`). Se preferir, divida esse valor por 30 para obter a taxa em Hyperions por segundo (`Hy/s`).
