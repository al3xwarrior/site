# Дата Ассеты

Дата-ассеты — это JSON-файлы, которые определяют поведение игрового процесса и лежат в основе основного контента Hytale. Они дают мощный, основанный на данных способ создавать и изменять игровые элементы — и всё это без необходимости писать код.

## Что такое дата-ассеты?

Дата-ассеты позволяют создавать или менять ключевые игровые элементы с помощью JSON-файлов конфигурации:

* Блоки: Определяют новые типы блоков и их свойства
* Предметы: Позволяют создавать уникальные предметы со своим
* **NPC**: Отвечает за настройку неигровых персонажей и их ИИ
* **Генерация мира**: Управляет генерацией мира
* **Таблицы добычи**: Определяют, какие предметы выпадают из различных источников

## Сравнение с Minecraft

Дата-ассеты аналогичны JSON файлам дата-паков Minecraft, но их возможности намного шире. Хотя дата-паки Minecraft несколько ограничены, архитектура Hytale, основанная на данных, означает, что основные элементы игры, такие как блоки, предметы и NPC, можно изменять с помощью файлов JSON.

Этот расширенный подход, основанный на данных, дает моддерам гораздо больше возможностей для настройки игры без необходимости погружаться в сложное программирование.

## Инструменты для разработки

Осознавая, что редактирование JSON вручную может быть достаточно утомительным и может быть подвержено ошибкам, Hytale планирует предоставить несколько инструментов, которые сделают работу с дата-ассетами более доступной:

### Редактор узлов / Нодовый редактор

В Hytale разрабатывают редактор узлов (Node Editor), чтобы сделать управление игровыми данными и ресурсами более удобным для пользователя:

* **Визуальный интерфейс**: Предоставляет графический способ работы с данными
* **Кастомизация фокусировки**: Особенно полезна для кастомизации NPC, взаимодействий и генерации мира
* **Интегрированный рабочий процесс**: работает напрямую в редакторе ассетов Hytale для создания единого набора инструментов.
* **Доступность**: упрощает взаимоотношения между сложными данными для понимания и изменения

### Дополнительные инструменты

Кроме редактора услов Hytale планирует поддерживать:

* **Визуальный JSON генератор**: Графические интерфейсы для создания JSON без написания синтаксиса
* **Редакторы на основе форм**: структурированные формы, которые автоматически генерируют соответствующий JSON
* **Множество вариантов рабочего процесса**: Разные инструменты для разных уровней подготовки и задач

## Сила и Простота

Эффективность дата-ассетов доказывается их простотой. Например, чтобы сделать в Hytale рапторов ездовыми, как сообщалось, требуется всего 4 строчки конфигурации в формате JSON — это показывает, насколько мощной и при этом доступной задумана эта система.

Этот баланс между мощностью и доступностью лежит в основе философии моддинга Hytale: предоставление инструментов профессионального уровня, для эффективного использования которых не требуются навыки программирования.

## Интеграция с другими системами

Дата-ассеты прекрасно сочетаются с другими подходами разработки Hytale:

* **Визуальное написание скрипта**: Можно ссылаться и обрабатывать данные, определенные в ресурсах данных
* **Java плагины**: Может считывать и изменять дата-ассеты для более сложных действий
* **Редактор Ассетов**: Предоставляет единый интерфейс для управления визуальными ассетами и их конфигурацией данных

Эта интеграция гарантирует, что данные не являются изолированными конфигурационными файлами, а скорее составной частью всеобъемлющей среды разработки Hytale.
